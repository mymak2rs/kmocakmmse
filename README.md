# K-MoCA/K-MMSE 인지장애 판별 홈페이지 개발

## 사용 기술
- Django (3.2.13) (https://www.djangoproject.com/)
- Bootstrap (v5.1) (https://getbootstrap.com/)
- python (3.9.12)

## 홈페이지 구성

- 메인 페이지 
- 환자 정보 입력 페이지
- 세부 항목 입력 페이지
- 결과 출력 페이지


## 기능

- 인지 장애 판별 결과 생성 (딥러닝, 머신러닝)

## Django 실행
``` 
    python kmocakmmse/manage.py runserver 
```
